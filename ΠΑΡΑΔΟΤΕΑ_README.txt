1. ΕΓΓΡΑΦΟ PDF (ΠΑΡΑΔΟΤΕΑ.html)
   ----------------------------
   Αρχείο: ΠΑΡΑΔΟΤΕΑ.html
   
   Περιεχόμενο:
   - Αριθμός ομάδας
   - Ονόματα και αριθμοί μητρώου συμμετεχόντων
   - Τίτλος εργασίας
   - Περιγραφή απαιτήσεων βάσης δεδομένων
   
   ΣΗΜΕΙΩΣΗ: Το αρχείο είναι σε μορφή HTML που μπορεί να μετατραπεί 
   εύκολα σε PDF. Για μετατροπή σε PDF:
   - Ανοίξτε το αρχείο στον browser
   - Πατήστε Ctrl+P (Print)
   - Επιλέξτε "Save as PDF" ως προορισμό εκτύπωσης
   
   ΣΗΜΑΝΤΙΚΟ: ΠΡΕΠΕΙ ΝΑ ΣΥΜΠΛΗΡΩΘΟΥΝ:
   - Ο αριθμός ομάδας
   - Οι αριθμοί μητρώου των συμμετεχόντων


2. ΑΡΧΕΙΟ SQL (Stoiximatiki_COMPLETE.sql)
   -------------------------------------
   Αρχείο: database/Stoiximatiki_COMPLETE.sql
   
   Περιεχόμενο:
   - Πλήρες SQL schema με όλους τους πίνακες
   - Λεπτομερείς σχόλια που υποδεικνύουν το ζητούμενο της εργασίας
   - Foreign Key constraints και περιορισμούς
   - Δειγματικά δεδομένα
   
   Χαρακτηριστικά:
   - Κάθε πίνακας έχει σχόλια που εξηγούν τον σκοπό του
   - Κάθε στήλη έχει περιγραφή
   - Οι σχέσεις μεταξύ πινάκων είναι καλά τεκμηριωμένες
   - Υποδεικνύονται τα ζητούμενα της εργασίας (π.χ. "ΖΗΤΟΥΜΕΝΟ 1", "ΖΗΤΟΥΜΕΝΟ 2")


3. ΚΩΔΙΚΑΣ ΔΙΑΔΙΚΤΥΑΚΗΣ ΕΦΑΡΜΟΓΗΣ
   -----------------------------
   Ο κώδικας της διαδικτυακής εφαρμογής βρίσκεται στον ριζικό φάκελο 
   και είναι ήδη οργανωμένος ως εξής:
   
   Δομή:
   ├── index.php                          # Κύρια σελίδα dashboard
   ├── login.php                          # Σελίδα σύνδεσης
   ├── logout.php                         # Script αποσύνδεσης
   │
   ├── Database pages/                    # Σελίδες διαχείρισης πινάκων
   │   ├── paikths.php                   # Διαχείριση Παικτών
   │   ├── ypallhlos.php                 # Διαχείριση Υπαλλήλων
   │   ├── stoixhma.php                  # Διαχείριση Στοιχημάτων
   │   ├── summetoxh.php                 # Διαχείριση Συμμετοχών
   │   ├── katathesi.php                 # Διαχείριση Καταθέσεων
   │   ├── logariasmos.php               # Διαχείριση Λογαριασμών
   │   ├── agonas.php                    # Διαχείριση Αγώνων
   │   ├── bonus.php                     # Διαχείριση Μπόνους
   │   └── paikths_bonus.php             # Διαχείριση Παικτών-Μπόνους
   │
   ├── Frontend/                          # Αρχεία CSS και JavaScript
   │   ├── style.css                     # Κύρια stylesheet
   │   ├── script.js                     # Κύρια JavaScript
   │   ├── particles.js                  # Particle effects
   │   ├── network.js                    # Network animations
   │   └── three-scene.js                # 3D scene (optional)
   │
   ├── config/                            # Αρχεία ρυθμίσεων
   │   ├── config.php.example            # Template για config.php
   │   └── auth.php                      # Authentication logic
   │
   └── assets/                            # Εικόνες και media
       ├── university_logo.png
       ├── xampos101.gif
       ├── mouxtaris.png
       └── dipapag.png

   Τεχνολογίες:
   - Backend: PHP 8.0+
   - Database: MySQL 8.0+
   - Frontend: HTML5, CSS3, JavaScript (ES6+)
   - Libraries: GSAP, Three.js, Lenis

   Λειτουργίες:
   - CRUD operations για όλους τους πίνακες
   - Admin authentication system
   - View-only mode για κανονικούς χρήστες
   - Modern UI με animations

================================================================================
                        ΟΔΗΓΙΕΣ ΧΡΗΣΗΣ
================================================================================

1. Για να δημιουργήσετε PDF από το HTML:
   - Ανοίξτε το ΠΑΡΑΔΟΤΕΑ.html στον browser
   - Πατήστε Ctrl+P
   - Επιλέξτε "Save as PDF"
   - Αποθηκεύστε ως "ΠΑΡΑΔΟΤΕΑ.pdf"

2. Για να εκτελέσετε το SQL:
   - Ανοίξτε το database/Stoiximatiki_COMPLETE.sql
   - Εκτελέστε το στο MySQL/MariaDB
   - Ή χρησιμοποιήστε το Stoiximatiki.sql για local development

3. Για να τρέξετε την εφαρμογή:
   - Ρυθμίστε το config/config.php με τα database credentials
   - Εκτελέστε: php -S localhost:8000
   - Ανοίξτε: http://localhost:8000
